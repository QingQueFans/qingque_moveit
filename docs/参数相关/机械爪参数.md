Franka Panda夹爪参数分析

finger_joint1 棱柱（直线） 0.0 到 0.04 0.2 20 左手指，主动控制

2. 几何参数

参数 值 [m] 说明
手指长度 0.0584 从手掌到指尖的距离
最大开口 0.08 两个手指各40mm，总计80mm
最小开口 0.0 完全闭合
手指间距 0.0 闭合时手指紧贴

3. 运动学关系

关键特征：

```
左手指标系：沿 +Y 方向移动
右手指标系：沿 -Y 方向移动（反向对称）
```

模仿关系：

```xml
<mimic joint="panda_finger_joint1" />
```

· 关节2完全模仿关节1的运动
· 只需要控制关节1，关节2自动跟随
· 确保两个手指同步对称运动

4. 物理连接

```
panda_hand（手掌）
    ├── panda_finger_joint1 ──► panda_leftfinger（左手指）
    └── panda_finger_joint2 ──► panda_rightfinger（右手指）
         └── 模仿关节1
```

7. 关键限制（重要！）

硬件限制：

1. 不能过度张开：超过80mm可能损坏
2. 不能强力闭合：对空闭合时力要小
3. 抓取深度有限：不能超过手指长度

控制特点：

1. 只需控制关节1：关节2自动跟随
2. 对称运动：确保平行夹持
3. 力/位置混合控制：抓取时需要

8. 实际应用配置

```python
GRIPPER_CONFIG = {
    # 物理限制（来自URDF）
    "physical": {
        "max_width": 0.08,      # 80mm
        "min_width": 0.0,       # 完全闭合
        "max_force_per_finger": 20.0,  # 20N
        "finger_length": 0.0584,       # 58.4mm
        "max_speed": 0.2,              # 0.2 m/s
    },
    
    # 控制配置（来自ROS2配置）
    "control": {
        "primary_joint": "panda_finger_joint1",
        "mimic_joint": "panda_finger_joint2", 
        "controller_name": "panda_hand_controller",
        "action_name": "gripper_cmd",
    },
    
    # 安全参数（经验值）
    "safety": {
        "safe_grasp_depth": 0.035,     # 35mm
        "default_grasp_force": 15.0,   # 15N（保守值）
        "object_min_size": 0.01,       # 10mm
        "object_max_size": 0.07,       # 70mm（留有余量）
    }
}
```

9. 特别提醒

1. URDF值保守：实际硬件能力更强（70N vs 40N）
2. 模仿关节：控制简单，但不能单独控制每个手指
3. 平行夹爪：适合规则物体，不适合复杂形状
4. 深度限制：抓取时要注意物体厚度

这些参数直接影响你的抓取成功率，特别是在计算抓取参数时需要严格遵守这些物理限制。
